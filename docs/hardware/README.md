# Hardware

用于记录真实硬件、接线、供电和风险项。所有实物信息以戴鹏林现场验证为准。

优先维护：

- `bom.md`：采购清单、到货状态、替代件、风险。
- `pinmap.md`：RDK、STM32、MDDS20、PCA9685、传感器接口编号。
- `power-tree.md`：12V 主电池、5V RDK、6V 舵机、电机驱动供电关系。
- `wifi_camera.md`：固定监控摄像头通过 WiFi/RTSP 接入 RDK X5 的网络方案。
- `wiring-log-template.md`：每次接线变更的记录模板。
- `photos/`：实物照片和标注说明。

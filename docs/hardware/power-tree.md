# 供电结构

整机建议采用 12V 主电池统一输入，分三路供电。

```text
12V 主电池
├── 主电源开关
├── 12V 直供 -> MDDS20 x2 -> 四个底盘电机
├── 12V -> 5V 8A 降压 -> RDK X5 / 小屏 / 低功耗 USB 外设
└── 12V -> 6V 5A UBEC -> PCA9685 舵机电源端 / 云台舵机
```

## 必须遵守

- 所有模块共地。
- RDK X5 使用独立 5V 支路。
- 舵机使用独立 6V 支路。
- 电机驱动吃 12V 主电源。
- 电机线与编码器线分开走线。
- 大电流线尽量短且粗。
- 传感器线远离电机驱动区域。
- 所有接口统一编号和标签。

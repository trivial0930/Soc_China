# 硬件集中在戴鹏林处的协作计划

## 协作原则

- 所有硬件集中放在戴鹏林处，避免频繁转移导致线缆、系统环境和装配状态混乱。
- 平日：曹鸿治、戎择辰以代码仓库/文档仓库提交为主；戴鹏林固定拉取最新代码并在真实硬件上验证。
- 周末：三人线下碰头，完成必须现场参与的接线、上电、底盘、电机和整机联调。
- 涉及电机、主电池、MDDS20、底盘运动的任务，必须有人现场看护，不能远程无人调车。

## 每人重点

### 戎择辰
远程主写RDK、ROS2、视觉、串口节点、日志系统；把代码提交到仓库，由戴鹏林下载验证。

### 戴鹏林
保管全部硬件；负责真实烧录、接线、上电、运行测试、上传日志和截图；周末带硬件参与集中联调。

### 曹鸿治
远程主做采购/验货表、BOM、接线记录、照片标注、答辩硬件说明；周末参与实际接线、拍摄和可靠性检查。

## 日常节奏

1. 戎择辰/曹鸿治当天提交代码或文档。
2. 戴鹏林晚上 git pull 后运行测试。
3. 戴鹏林把日志、截图、串口输出或失败现象发到群里。
4. 第二天远程成员根据反馈继续修改。
